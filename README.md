Spring Boot, HtmlUnit
---------------------------------------------------

Tools used:
-----------

* Maven 3.5
* Spring Boot 1.5.8-RELEASE
* SeleniumHQ 3.7.1
* ngwebdriver 1.0 (https://github.com/paul-hammant/ngWebDriver)

**Run**

    $ mvn spring-boot:run
    $ /developer/chromedriver_win32/chromedriver
    $ http://localhost:8080/crawler/cetip/000.000.000-00/0/0

**Tests**

    $ mvn test
